> Benchmarking dns server


## Install

```sh
$ npm install --save dnsperf
```


## Usage

```js
var dnsperf = require('dnsperf');

dnsperf('Rainbow');
```

```sh
$ npm install --global dnsperf
$ dnsperf --help
```
